HTML & CSS Cheatsheet

HTML Document Tags

Tag Name Description
<html> HTML tag Creates a basic HTML document.
<head> HTML head Used to enclose the head of the HTML document.
<body> HTML body Used to enclose the body of the HTML document.
<header> HTML header Used to enclose an HTML document's header content.
<nav> HTML nav Used to enclose navigation links
<ol> HTML Ordered List tag This tag contains an ordered list.
<li> List Item tag This tag contains a list item from an ordered or unordered list.
<ul> HTML Unordered List An HTML list in no particular order.
<li> List Item tag This tag contains a list item from an ordered or unordered list.
<footer> Table footer Used to enclose the footer of the HTML document.

HTML Tables

Tag Name Description
<table> Table Creates a table.
<thead> Table Head Used for the table's header cells.
<th> Table Heading Used in the table head only for headings.
<tbody> Table Body Used to enclose a table's body content.
<tr> Table Row Table row container.
<td> Table Data Table data container.
<tfoot> Table Foot Contain's the table's footer content.
<colspan> Column Span Defines how many columns a td element should span.
<colspan> Row Span Defines how many rows a td element should span.
